Privacy Policy

This page describes the privacy practices of the Sabis Not Hesaplayıcı Chrome extension. The purpose of the extension is to let users locally calculate grade averages on grade tables within Sabis OBS pages. Below you will find which data is processed, why the requested permissions are needed, and what user control options exist.

Quick Summary

The extension does not collect personal data and sends no data to external servers. All calculations and data processing occur locally in the user's browser only on the visited Sabis pages. The extension stores only a single boolean setting (`extensionEnabled`) inside chrome.storage.local.

What data is processed?

  • Page DOM data: The extension reads only the grade tables on the currently open Sabis OBS page to perform calculations. This reading occurs via the DOM within the browser; the data is not transmitted anywhere by the extension.
  • Local setting: A single boolean value is written to chrome.storage.local to store the on/off status of the extension. This value is not considered personal data.
  • User inputs: The user may manually enter grades into some empty grade cells; these inputs are used only on that page for calculation and are not sent to a server.

Why are permissions requested?

  • activeTab: Needed to access the active tab after user interaction in the popup and, if required, to refresh the tab to apply the extension state. This access occurs only through direct user action.
  • Host permission (https://obs.sabis.sakarya.edu.tr/*): The extension runs only on Sabis OBS pages. This host permission is required to locate grade tables in the DOM and perform local calculations. The extension does not access other sites.
  • scripting: Used to make small DOM changes such as adding input fields or styles. These changes are limited to the page the user is visiting and are triggered by the user or automatically if the extension is enabled.
  • storage: Required to store the on/off state of the extension (only one boolean).
Note: This extension does not execute any remote JavaScript or WebAssembly (Wasm). A Font Awesome CSS CDN may have been used for icons in the popup view; this static CSS file does not affect logic or user data. If desired, the icons can be bundled locally so it works without a CDN.

Data sharing and third parties

The extension does not send user data to third-party servers. Grades entered by the user, calculated averages, or extension settings (only a boolean) are not collected or shared.

Third-party links and static assets

Some visual assets used in the popup (e.g. Font Awesome CSS) may be loaded via a CDN. This CSS file is only for icon display and does not execute logic or transmit user data. The "Buy Me a Coffee" button in the popup directs the user to buymeacoffee.com; this navigation is a deliberate user action and the extension does not automatically share data.

Important: When navigating to external sites, those sites may collect data according to their own privacy practices (e.g. cookies, analytics). The data practices of such external sites are outside the control of this extension.

Data sharing with third parties

By default no user data is shared with third parties. The only exception is if the user manually clicks an external link (such as the BuyMe link); in that case the target site is visited and may apply its own data policies.

Security

The extension does not store sensitive keys or secrets. Key files such as the .pem used during packaging are not kept in the repository root; the developer stores these keys locally and securely. Source code and package contents are open to inspection.

Data retention and deletion

The extension stores only a single boolean setting inside chrome.storage.local. This data is deleted in these situations:

  • When the user uninstalls the extension, all related chrome.storage.local data is removed.
  • If the user manually clears extension settings or clears browser data, the related data may be removed.
If you specifically want your data fully removed, please send a request to ozdemirosmantahir@gmail.com; after receiving your request we will provide the necessary guidance and confirmation.

Analytics and tracking

No third-party analytics SDKs or telemetry collection code exist in this extension package. On the developer side no external analytics service is active for automatic usage data or error reporting.

Policy changes

This privacy policy may be updated when necessary. For material changes this page will be updated and the last update date will be indicated in the area above. In case of changes, important updates may be communicated through the provided contact address.

User rights and controls

  • The user can disable the extension at any time via the popup (when the toggle is off the extension performs no actions on the page).
  • If desired the user can completely remove the extension from the browser's extensions list.
