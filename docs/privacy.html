Privacy Policy

Last Updated: September 24, 2025

Introduction

The Sabis Average Calculator ("the extension") is designed to calculate grade averages on the Sakarya University Information System (SABİS) pages. Your privacy is important to us. This policy explains what data the extension accesses and how it is used.

Single Purpose

This extension has a single purpose: to read the grade table data (such as course names, credits, and grades) exclusively on https://obs.sabis.sakarya.edu.tr/, and then locally calculate and display the semester and overall grade point average within the user's browser.

Data Collected and Used

To perform its function, the extension accesses data classified as "Website Content." This data includes:

  • Course information, credit values, and grades from the SABİS transcript page at https://obs.sabis.sakarya.edu.tr/

This data is used exclusively for real-time grade average calculation and is never saved, stored, transmitted to servers, or shared with any third parties. All calculations happen locally within your browser and the data is only processed momentarily.

Extension Settings Storage

The extension uses Chrome's built-in storage API to remember only one simple setting: whether the extension is enabled or disabled (on/off). This setting:

  • Contains no personal information whatsoever
  • Is stored securely using Chrome's official storage system
  • Can sync across your Chrome browsers when signed in (optional Chrome feature)
  • Can be deleted by uninstalling the extension

Justification for Minimal Permissions

This extension requests only the minimal permissions necessary for its single purpose:

  • storage Permission: Required to save your on/off preference for the extension. This uses Chrome's secure storage system and contains no personal data - only a simple boolean value (true/false) indicating if the extension should be active.
  • Content Scripts on https://obs.sabis.sakarya.edu.tr/*: The extension operates exclusively on SABİS pages to read grade tables and display calculated averages. It cannot and does not access any other websites or data.

Important: We deliberately avoided requesting permissions like activeTab, scripting, or broad host permissions to minimize privacy impact and ensure Google Chrome Web Store approval.

Data Sharing and Security

User data (including website content) is not sold, transferred, or used for any purpose other than the approved single purpose (grade calculation). Your data belongs to you and remains within your browser.
