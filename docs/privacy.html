Privacy Policy

Last Updated: September 23, 2025

Introduction

The Sabis Average Calculator ("the extension") is designed to calculate grade averages on the Sakarya University Information System (SABİS) pages. Your privacy is important to us. This policy explains what data the extension accesses and how it is used.

Single Purpose

This extension has a single purpose: to read the grade table data (such as course names, credits, and grades) exclusively on https://obs.sabis.sakarya.edu.tr/, and then locally calculate and display the semester and overall grade point average within the user's browser.

Data Collected and Used

To perform its function, the extension accesses data classified as "Website Content." This data includes:

  • Course information, credit values, and grades from the SABİS transcript page.

This data is used momentarily for the sole purpose of calculation and is absolutely not saved, stored, sent to our servers, or shared with any third parties. All calculations happen locally on your browser.

Justification for Permissions

  • Host (https://obs.sabis.sakarya.edu.tr/*) and scripting Permissions: These permissions are mandatory for the extension to operate on the specified SABİS page, read the grade table, and display the calculated average. The extension does not run or read data on any other website.
  • storage Permission: This is used to save the user's preference for the extension's state ("on" or "off") in your browser's local storage. No personal data is stored here.
  • activeTab Permission: This permission allows the extension to gain temporary access to the active tab when the user clicks the extension's icon. This is a secure method that is only triggered by user interaction.

Data Sharing and Security

User data (including website content) is not sold, transferred, or used for any purpose other than the approved single purpose (grade calculation). Your data belongs to you and remains within your browser.
